# CiviMember Roles Sync

Synchronize CiviCRM Contacts with Membership Status to a specified Drupal
Role both automatically and manually.


## Table of Contents

- [Installation](#installation)
- [Configuration](#configuration)
- [Maintainers](#maintainers)
- [Supporting Organizations](#supporting-organizations)


## Installation

- Install as usual. You can find further information on how to install contributed
modules in Drupal 8 [here](https://www.drupal.org/docs/8/extending-drupal-8/installing-contributed-modules-find-import-enable-configure-drupal-8).


## Configuration

- Set the permissions to establish which roles can manage this module.
- Go to `/admin/config/civicrm/civicrm-member-roles` to set the roles
  equivalences.
